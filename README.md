# soviet-cinema-data

Структурированные данные проекта **Soviet Bloc Cinema** — открытого исследовательского портала о кино СССР и стран социалистического лагеря XX века.

## Что здесь лежит

| Директория       | Содержимое                                          |
|------------------|-----------------------------------------------------|
| `films/`         | Фильмы (`<slug>.yaml`)                              |
| `people/`        | Режиссёры, актёры, сценаристы, операторы и др.      |
| `studios/`       | Киностудии                                          |
| `motifs/`        | Визуальные и смысловые мотивы                       |
| `references/`    | Связи фильм → фильм и фильм → произведение           |
| `vocabularies/`  | Контролируемые словари (страны, жанры, роли и т.д.) |
| `docs/adr/`      | Architecture Decision Records                        |
| `reports/`       | Отчёты импорта и валидации (не коммитятся целиком)  |

Полная схема и конвенции — в `../CLAUDE.md` корня проекта.

## Источники

Часть фактов перенесена из:

- **Wikidata** — CC0;
- **Wikipedia** — CC BY-SA 4.0.

Идентификаторы внешних источников указываются в поле `external_ids` соответствующих записей.
Изображения постеров и кадров в репозитории **не хранятся** (см. CLAUDE.md, раздел «Работа с изображениями»).
